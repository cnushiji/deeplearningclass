1. Problem formulation

Consider the following time series problem

{x1(n)=x1(n−1)+sin(x1(n−1))+w1(n),x2(n)=x1(n−1)+cos(x2(n−1))+w2(n),y(n)=x12(n)+x22(n)+v(n)\left \{ \begin{aligned} x_1(n) &= x_1(n-1) + sin(x_1(n-1)) + w_1(n), \\ x_2(n) &= x_1(n-1) + cos(x_2(n-1)) + w_2(n), \\ y(n) &= \sqrt {x_1^2(n)+x_2^2(n)} + v(n) \end{aligned} \right .⎩⎨⎧​x1​(n)x2​(n)y(n)​=x1​(n−1)+sin(x1​(n−1))+w1​(n),=x1​(n−1)+cos(x2​(n−1))+w2​(n),=x12​(n)+x22​(n)​+v(n)​
where w1w_1w1​, w2w_2w2​ and vvv are mutual independent standard Gaussian white noises.

2. Goal

Suppose we don't know what the system dynamic, we have only a dataset Dtrain={x(i),y(i)}i=1ND_{train}=\{ x^{(i)}, y^{(i)} \}_{i=1}^NDtrain​={x(i),y(i)}i=1N​, we want to make predictions for unkonwn trajectories Dtest={x(i),y(i)}i=1mD_{test}=\{ x^{(i)}, y^{(i)} \}_{i=1}^mDtest​={x(i),y(i)}i=1m​.

3. Method

We use RNN to solve this problem.

3.1 Model

We use a sample RNN model with only one layer, bidirectional. The number of hidden units are 10. We use MSE loss as performance metric for this task.

class my_rnn_net(nn.Module):
    def __init__(self):
        super(my_rnn_net,self).__init__()
        self.rnn = nn.RNN(input_size=2,hidden_size=10,num_layers=1,nonlinearity='tanh',bias=True,batch_first=True,bidirectional=True)

        self.fc = nn.Linear(in_features=20,out_features=1)

    def forward(self,seq):
        H,hn = self.rnn(seq)
        Y = self.fc(H)

        return Y

3.2 Experiment setting
